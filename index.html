Siempre distribución z
Fórmula: p̂ ± z·√(p̂·q̂/n)   donde q̂ = 1−p̂

donde p̂ = (x₁+x₂)/(n₁+n₂)

📊 IC para la media (μ)
▾
Caso 1 — σ conocida (z)
x̄ ± z · (σ/√n)
n puede ser grande o pequeña
Caso 2 — σ desconocida, n>30 (z)
x̄ ± z · (s/√n)
Usamos s en lugar de σ
Caso 3 — σ desconocida, n≤30 (t)
x̄ ± t · (s/√n)
gl = n − 1
📈 IC para la proporción (P)
▾
Fórmula única (z)
p̂ ± z · √(p̂·q̂/n)
q̂ = 1 − p̂
ANÁLISIS DEL INTERVALO [A, B]
Si A<0 y B<0 → diferencia significativa, P₁ < P₂
Si A>0 y B>0 → diferencia significativa, P₁ > P₂
Si A<0 y B>0 → NO hay diferencia, P₁ = P₂
🔀 IC para μ₁−μ₂ (4 casos)
▾
Caso 1 — σ₁,σ₂ conocidas (z)
(x̄₁−x̄₂) ± z·√(σ₁²/n₁ + σ₂²/n₂)
Caso 2 — σ desconocidas, n₁+n₂>30 (z)
(x̄₁−x̄₂) ± z·√(s₁²/n₁ + s₂²/n₂)
Caso 3 — n₁+n₂≤30, σ₁²=σ₂² (t, Sp)
(x̄₁−x̄₂) ± t·Sp·√(1/n₁+1/n₂)
Sp = √[(n₁−1)s₁²+(n₂−1)s₂²] / (n₁+n₂−2)
gl = n₁+n₂−2
Caso 4 — n₁+n₂≤30, σ₁²≠σ₂² (t, Welch)
(x̄₁−x̄₂) ± t·√(s₁²/n₁+s₂²/n₂)
gl = fórmula de Welch-Satterthwaite
ANÁLISIS DEL INTERVALO [A, B]
Si A<0 y B<0 → μ₁ < μ₂ (diferencia significativa)
Si A>0 y B>0 → μ₁ > μ₂ (diferencia significativa)
Si A<0 y B>0 → μ₁ = μ₂ (NO hay diferencia)
🧪 Prueba de hipótesis — 5 pasos
▾
Paso 1 — Plantear hipótesis
H₀: = (hipótesis nula)
Hₐ: ≠ / > / < (hipótesis del investigador)
Paso 2 — Distribución
Normal z · t-Student · Chi² · F
Paso 3 — Nivel α y valor crítico
Marcar zona de rechazo y aceptación en la gráfica
Paso 4 — Estadística de prueba
Calcular Zc, tc, χ²c, Fc con los datos de la muestra
Paso 5 — Decisión
Ubicar el estadístico en la gráfica → zona de rechazo o aceptación

Si p-valor < α → se rechaza H₀

IC · Diferencia de medias
Nicotina en cigarrillos (Ejemplo 1 del profesor)
Marca 1: {24,26,25,22,23} · Marca 2: {27,28,25,29,26}. ¿Existe diferencia en el contenido medio de nicotina? IC al 95%, varianzas iguales.
Datos: x̄₁=24, x̄₂=27, s₁=s₂=1.58114, n₁=n₂=5
Caso: σ desconocidas, n₁+n₂=10≤30, varianzas iguales → Caso 3 (t)
gl: v = n₁+n₂−2 = 5+5−2 = 8 → t₀.₀₂₅,₈ = 2.30600
Sp: √[(4·2.5+4·2.5)/8] = √2.5 = 1.58114
IC: (24−27) ± 2.306·1.581·√(1/5+1/5) = −3 ± 2.306
A = −5.306 · B = −0.694
Conclusión: A<0 y B<0 → SÍ hay diferencia. Marca 1 tiene menos nicotina que Marca 2.
PH · Media
Salario trabajadores Pasto (Ejemplo 1 PH)
El alcalde afirma que el salario promedio supera 4.2 sm. Muestra de 8 trabajadores: {3.8,5.7,6.2,4.3,5.8,6.4,4.5,5.5}. α=5%.
Paso 1: H₀: μ=4.2 · Hₐ: μ>4.2 (cola derecha)
Paso 2: t-Student (σ desconocida, n=8≤30) · gl=7
Paso 3: α=0.05, tcrit=t₀.₀₅,₇ = 1.89458 (cola derecha)
Paso 4: x̄=5.275, s=0.9528 · tc = (5.275−4.2)/(0.9528/√8) = 3.1909
Paso 5: tc=3.19 > tcrit=1.895 → ZONA DE RECHAZO de H₀
Conclusión: El alcalde SÍ tiene razón. El salario promedio supera los 4.2 sm.
PH · Proporción
Internet en la Udenar (Ejemplo 2 PH)
El rector afirma que el 85% de estudiantes tienen internet. Muestra de 30: 19 responden "sí". Nivel de confianza 90%.
Paso 1: H₀: P=0.85 · Hₐ: P≠0.85 (bilateral)
Paso 2: Distribución normal estándar (z)
Paso 3: α=0.10 · zcrit = ±1.6449
Paso 4: p̂=19/30=0.6333, q̂=0.3667 · zc = (0.6333−0.85)/√(0.6333·0.3667/30) = −4.022
Paso 5: |zc|=4.022 > 1.645 → ZONA DE RECHAZO de H₀
Conclusión: El rector NO tiene razón. El 85% es incorrecto.
IC · Proporción
Embarazadas en Udenar (Ejemplo 1 IC-P)
De 150 estudiantes, el 15% dio positivo en prueba de embarazo. Calcular IC al 90% para la verdadera proporción.
Datos: n=150, p̂=0.15, q̂=0.85
Confianza: 90% → z=1.6448
SE: √(0.15·0.85/150) = 0.02915
IC: 0.15 ± 1.6448·0.02915 = 0.15 ± 0.04795
A = 10.21% · B = 19.79%
Conclusión: Con 90% de confianza, entre el 10.21% y el 19.79% de estudiantes de Udenar están embarazadas.
PH · Dif. de medias
Ácido ascórbico — fumadoras vs no fumadoras
No fumadoras: n₁=24, x̄₁=0.9158, s₁=0.2146. Fumadoras: n₂=7, x̄₂=0.9471, s₂=0.4134. ¿Las fumadoras tienen mayor nivel? α=5%, varianzas iguales.
Paso 1: H₀: μ₁=μ₂ · Hₐ: μ₁<μ₂ (cola izquierda)
Paso 2: t-Student, n₁+n₂=31≤30? No, 31>30 pero se suponen varianzas iguales → Caso 3, gl=29
Paso 3: α=0.05, tcrit = −1.699 (cola izquierda, gl=29)
Sp: √[(23·0.2146²+6·0.4134²)/29] = 0.2681
Paso 4: tc = (0.9158−0.9471)/(0.2681·√(1/24+1/7)) = −0.2717
Paso 5: tc=−0.272 > −1.699 → ZONA DE ACEPTACIÓN
Conclusión: NO se puede afirmar que las fumadoras tienen mayor nivel de ácido ascórbico.
